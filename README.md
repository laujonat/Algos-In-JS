# Javascript Practice

## Project Setup

```sh
 $ npm install
 $ npm run start:dev
 $ npm run start
```

## Dependencies

- Flow Type
- Webpack@^4.1.7
- Express@^4.0.0

## JSON Template

In `module.exports`, specify problem properties in JSON:

- Problem Type: <{"Sorting" | "Recursion" | "Trees" | "Graphs" | "Dynamic Programming"}: String>
- Name: <String>
- Filepath: <String>
- Exports: Array<String>
- Input Params: Array<mixed>
