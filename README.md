## Create Mongodb Account. See documentation.

Create a new db collection. Create a `.env` and fill in the following fields.

```
 MONGODB_BASE=mongodb+srv://
 MONGODB_URL=<dburl>
 MONGODB_USER=<user>
 MONGODB_PASSWORD=<password>
```
